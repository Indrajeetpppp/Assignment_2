# Python Assignment

## Task 1: Even/Odd Checker
- The program asks the user to enter a number.
- It uses the modulus operator (`%`) to check if the number is even or odd.
- The result is printed to the screen.

## Task 2: Sum of Numbers (1 to 50)
- The program calculates the sum of numbers from 1 to 50.
- A variable `total` is initialized to 0.
- In each loop iteration, the current number is added to `total` (this is called accumulation).
- After the loop finishes, the final sum (1275) is printed.

### How to Run
1. Save the code in a file named `Task-1.py  &  Task-2.py`.
2. Open a terminal or command prompt.
3. Run the program.
